# 功能
   将选中的汉字翻译成英语，尽量避免为了给变量起一个有意义的名字而冥思苦想 (ノ｀ー´)ノ・・・~~┻━┻
   
# 安装
方式一. 下载工程中的ZHTranslationPlugin.jar导入Android Studio  
方式二. 在Android Studio插件中心搜索 ZHTranslationPlugin  
   
# 使用
   1.选中待翻译的汉字  
   2.按ALT+X快捷键呼出翻译列表  
        2.1 ALT+X  
        2.2 键盘上下键选择待选文案  
        2.3 entry 确定  
        2.4 ALT+Q关闭页面
